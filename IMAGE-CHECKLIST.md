## üé• OPTIONAL BUT HIGH-IMPACT: HERO VIDEO

### Hero Background Loop
**Filename:** `hero-loop.mp4`
**Ideal Content (15-30 seconds):**
1. Garage door opening (3-5 seconds)
2. Walking through interior showing kitchen (5-8 seconds)
3. Quick cuts of amenities - pickleball, fire pit, dog park (8-10 seconds)
4. End on exterior shot or logo (2-3 seconds)

**Specs:**
- Resolution: 1920x1080 (Full HD)
- Format: MP4 (H.264 codec)
- File Size: Under 5MB (compress heavily)
- Loop-friendly (ends smoothly so it can repeat)
- No audio needed (video is muted on autoplay)

**Can be shot on iPhone!**
- Use golden hour lighting
- Stabilize footage (gimbal or steady hands)
- Avoid shaky shots
- Keep clips short and dynamic

## üõ†Ô∏è IMAGE OPTIMIZATION TOOLS

**Before uploading, compress images:**

### Online (Free):
- TinyPNG.com (best for JPG/PNG)
- Squoosh.app (Google's tool)
- Compressor.io

### Desktop (Mac):
- ImageOptim (free, drag & drop)
- Preview app (built-in, export with lower quality)

### Desktop (Windows):
- RIOT (Radical Image Optimization Tool)
- FileOptimizer

**Goal:** Keep images under recommended file sizes without losing visual quality.

## üìã REPLACEMENT PROCESS

### Step 1: Gather Photos from Joe's B-Roll
- Request all interior shots
- Request all amenity shots
- Request exterior shots
- Request any drone footage

### Step 2: Select Best Shots
- Golden hour preferred
- People in shots add life (but not required)
- Show scale and space
- Clean, professional look

### Step 3: Edit/Crop if Needed
- Crop to recommended dimensions
- Adjust lighting/contrast if needed
- Keep natural look (don't over-edit)

### Step 4: Optimize File Sizes
- Use compression tools above
- Check file size before uploading

### Step 5: Rename Files to Match
- Use exact filenames from this checklist
- Example: `hero-corporate.jpg` (not `IMG_1234.jpg`)

### Step 6: Upload to Project
```bash
# In your project folder:
cd palecero-landing/images/
# Copy your optimized images here

# Then commit and push:
git add images/
git commit -m "Add property images"
git push
```

### Step 7: Verify on Live Site
- Check all images load
- Verify quality looks good
- Test on mobile devices
- Confirm fast loading
