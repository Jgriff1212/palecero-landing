# Images Directory

## Place Your Property Photos Here

This folder should contain all images for your landing page.

### Required Images:

**Hero Images (3):**
- `hero-corporate.jpg` - For corporate relocator audience
- `hero-faculty.jpg` - For faculty/staff audience  
- `hero-lifestyle.jpg` - For young professionals audience

**Feature Images (6-8):**
- `placeholder-garage.jpg` - Private garage interior
- `placeholder-kitchen.jpg` - Modern kitchen
- `placeholder-living.jpg` - Spacious living room
- `placeholder-bedroom.jpg` - Bedroom with natural light
- `placeholder-pickleball.jpg` - Pickleball court
- `placeholder-dogpark.jpg` - Dog park
- `placeholder-firepit.jpg` - Fire pit area
- `placeholder-trail.jpg` - Walking trail (optional)

**Other:**
- `placeholder-map.jpg` - Location map
- `palecero-logo.png` - Property logo (PNG with transparency)

### Image Specifications:

**Hero Images:** 1920x1080px, under 500KB each
**Feature Images:** 800x600px, under 300KB each
**Logo:** PNG format with transparent background

### Quick Start:

1. Get photos from videographer Joe's B-roll
2. Optimize images (use TinyPNG.com)
3. Rename to match filenames above
4. Copy into this folder
5. Commit and push to GitHub
6. Vercel auto-deploys!

See **IMAGE-CHECKLIST.md** in the root folder for detailed instructions.
